Mi primera página con CSS

Experimentando con los estilos nuevos para mis proyectos con CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. Permite separar el contenido de la estructura, facilitando el diseño y la personalización de las páginas web.

Conceptos básicos de CSS

  • Selector: Define a qué elementos HTML se aplicarán los estilos.
  • Propiedad: Especifica qué aspecto del elemento se va a modificar (color, tamaño, margen, etc.).
  • Valor: Indica el valor que se asignará a la propiedad.
